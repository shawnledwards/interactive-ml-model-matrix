# ML Model Selection Matrix

An interactive reference tool for mapping business use cases to the right machine learning model.

## Launch

**[Open the ML Model Selection Matrix](https://shawnledwards.github.io/interactive-ml-model-matrix/)**

## What It Does

Maps 20 real-world business use cases across 6 industries to the most appropriate ML models, with guidance on why each model fits and what caveats to consider.

**Industries covered:** Retail · Healthcare · Finance · Energy · Marketing · Operations

**Models compared:**
| Model | Best for |
|---|---|
| Linear Regression | Predicting a continuous number (revenue, price, dose) |
| Logistic Regression | Binary classification (churn, default, fraud) |
| Decision Tree / RF | Non-linear relationships, mixed variable types |
| Neural Network | High-dimensional or unstructured data (text, images) |
| Clustering (K-Means) | Unsupervised segmentation (no labelled target) |
| Time Series | Temporal patterns (demand, load, returns) |

## Features

- Filter by industry
- Search across use cases, industries, and outputs
- Click any row for a detailed breakdown: inputs, model recommendations, and key considerations

## Usage

No installation needed — open the link above in any browser. To run locally:

```bash
open index.html
```
